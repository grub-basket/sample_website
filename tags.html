Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell
div division divides parts of the page
header header a place to nest your headers? Not really sure.
ul Unordered List No priority; no numbers.
ol Ordered List List based on Priority that is numbered.
li list include a list item (ordered or unordered)
blocquote block quotation show formatted quotation
br break enter line break

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong strong make strong text <strong>at least a billion people</strong> at least a billion people
a anchor make hyperlink <a href="http://learnenough.com/">Learn Enough</a> Learn Enough
img image include an image <img src="http://bit.ly/1MZAFuQ" alt="Michael Hartl">
Code code format as source code <code>header</code> header
span span acts like div, but is an inline element <span>Call me maybe.</span> Call me maybe.

Document Tags

Tag Definition Usage
html The <html> tag tells the browser that this is an HTML document. Informs the browser of the document type, which is HTML.
head The <head> element is a container for all the head elements. The <head> element can include a title for the document, scripts, styles, meta information, and more.
body The <body> tag defines the document's body. The <body> element contains all the contents of an HTML document, such as text, hyperlinks, images, tables, lists, etc.
title The <title> tag is required in all HTML documents and it defines the title of the document. The <title> tag is useful for favorites, describing a tab, and is used in search engine results.
meta Metadata is data (information) about data. Meta elements are typically used to specify page description, keywords, author of the document, last modified, and other metadata.

The table above contains information from w3schools - tags.
